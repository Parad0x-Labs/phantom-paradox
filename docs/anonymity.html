// layer breakdown

1. VAULT LAYER

User deposits into LP-funded vault. The vault is pre-funded by liquidity providers, so your deposit mixes with existing funds.

2. POLTERGEIST LAYER

Ghost wallets inject fake transactions. Real transfers are hidden among 10-100+ decoy transfers. Impossible to distinguish signal from noise.

3. HYDRA LAYER

1000+ rotating vault addresses (shards). Funds scatter across shards before settlement. Addresses rotate every epoch â€” by the time they're flagged, funds have moved.

4. MERKLE LAYER

Batch compression. 1000+ transactions become 1 Merkle root on-chain. No individual transfers visible â€” only the cryptographic commitment.
