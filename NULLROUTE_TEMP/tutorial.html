Phantom Devnet Setup

// switch wallet to test network

DEVNET ONLY

Devnet SOL has no real value. It's free test currency for developers.

1 Open Phantom Wallet
Make sure you have the Phantom browser extension installed. Click the Phantom icon in your browser toolbar.
2 Open Settings
In the Phantom popup, click the Settings icon (gear icon, bottom-left corner).
3 Enable Developer Settings
Scroll down to find "Developer Settings" and toggle "Testnet Mode" ON.
4 Switch to Devnet
Go to Settings -> "Change Network" -> Select "Solana Devnet"
Your wallet will show "Devnet" badge and balance resets to 0 (this is normal)
5 Get Test SOL
Return to NULLROUTE and use the Devnet Faucet to get free test SOL for transactions.

Setup Complete

Your Phantom wallet is now connected to Solana Devnet. You can test NULLROUTE transfers with free Devnet SOL.
